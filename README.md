Вариант 10:
Реализовать возможность перемещения и поворота произвольной HTML страницы (через iframe) средствами CSS 2D transforms 
(при помощи управляющих кнопок и при помощи мыши).

Работает в браузерах Chrome, Safari и Opera.
Перемещение осуществляется с помощью клавиш-стрелок, повороты с помощью нажатия на левую кнопку мыши и кнопку колесика.
